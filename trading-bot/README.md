# Hermes Trading Bot

Self-improving BTC/USDT paper trading agent. Runs 24/7 on Railway, logs trades to Supabase, and calls Claude AI every 5 trades to evolve its own strategy.

**Tech:** Python · FastAPI · ccxt · Anthropic SDK · Railway
